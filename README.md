Object Detection을 이용한 식재료 인식 및 응용 애플리케이션
==============================================================
<br>

## 1. HW 개발환경
- OS : Android Oreo
- Processor : 엑시노스 7 Octa
- RAM : 3GB
- Display : 2560 x 1440
<br>

## 2. SW 개발환경
- Server : Apache Tomcat(ver. 8.5.34)
- DBMS : MySQL(ver. 8.0.12)
- Android IDE : Android Studio(ver. 3.1.4)
- Deep Learning : Tensorflow(ver. 1.15.0) / Yolo v3
<br>
